# MSc Research Platform Overview

The `stemgen-platform` repository is the implementation platform for the MSc research prototype and evaluation. It is not the dissertation manuscript repository.

The platform supports research around AI-generated STEM visual explanations, animations, and simple narration for Tanzanian primary education.

## Project Framing

Research intent leads to prototype/platform implementation, evaluation workflow, evidence generation, and then dissertation write-up.

This project is responsible for:

- prototype and platform implementation,
- evaluation workflow support,
- evidence generation,
- reproducible experiment and result artifacts.

This project is not primarily responsible for:

- literature review writing,
- UDSM formatting,
- dissertation chapter editing.

## Review Questions

- Which research objective does the work support?
- Which evaluation activity does it enable?
- What evidence will it produce?
- Is it necessary for the dissertation, or is it overengineering?
